# Research state model

Allowed states:

`proposed Ôćĺ planned Ôćĺ running Ôćĺ completed Ôćĺ checked Ôćĺ validated Ôćĺ accepted`

Alternative terminal states: `rejected`, `superseded`.

- completed: execution ended and artifacts exist;
- checked: structural and procedural checks passed;
- validated: scientific and methodological validation passed;
- accepted: the result is judged relevant and usable for the research decision.

Transitions require recorded evidence and, where configured, human approval. A rejected result remains in the audit trail.
