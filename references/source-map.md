# Inspiration and catalog source map

TsaoSciResearcher is an original implementation. No upstream source code or prompt corpus is vendored. Public repositories informed capability discovery and architectural comparison only. See `THIRD_PARTY.md` for license boundaries and canonical links.

The uploaded 322-entry AI-for-Science workbook is treated as requirements metadata. Release v0.5.2 preserves all 322 catalog slugs and names as machine-readable contracts: 158 general research contracts and 164 computation/engineering domain contracts. The records were rewritten into TsaoSciResearcher's schema and do not copy executable upstream skills.

The 32 listed scientific engines are integration targets. Their names in a capability contract do not mean the engine, database, model or instrument is bundled or has executed.
