# Peer review

Review in layers: question significance, novelty, design validity, method reproducibility, data quality, statistics, figure integrity, claim support, limitations and reporting compliance. Separate fatal validity problems from remediable clarity issues.

A response-to-reviewers document must answer each comment, state the action, identify manuscript locations and disclose when a request cannot be satisfied. The response must not claim new calculations or experiments were run unless artifacts exist and passed validation.
