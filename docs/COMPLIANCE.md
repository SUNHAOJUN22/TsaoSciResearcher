# Design compliance summary

**Release:** v0.5.2

TsaoSciResearcher satisfies the uploaded design as a research-method, evidence, validation and orchestration system.

## Complete

- single-entry progressive router;
- 15 gated workflows;
- 322 exact named AI-for-Science catalog contracts plus 18 runtime/core contracts;
- unified design-compatible `.tsao-research/` state and registries;
- project, evidence, claim, citation, figure, export, schema, structure and release validation;
- Codex, Claude Code and Open Agent Skills installation;
- Windows/Linux/macOS CI;
- deterministic safe ZIP release;
- original Apache-2.0 implementation with documented upstream boundaries.

## Deliberately delegated

External databases, DFT/MD/FEM/CFD/process solvers, HPC schedulers, laboratory instruments and Office renderers are not bundled. They require a real host tool or a checksummed computation handoff. Software completion is not scientific acceptance.

See `docs/README_ARCHITECTURE_MAPPING.md` for the detailed design-to-test chain.
