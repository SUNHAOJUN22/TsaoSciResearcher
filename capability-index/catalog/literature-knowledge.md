# 文献与知识工程

- `literature-search-strategist` — **文献检索策略**: 生成关键词、同义词、布尔检索式和数据库组合
- `systematic-review-builder` — **系统综述**: 执行PRISMA式检索、筛选和证据汇总
- `paper-relevance-screener` — **论文相关性筛选**: 按研究问题、对象、方法和证据层级筛选论文
- `paper-quality-assessor` — **论文质量评估**: 评价实验设计、样本量、统计和结论可信度
- `paper-structure-parser` — **论文结构解析**: 抽取摘要、方法、结果、讨论、图表和补充材料
- `scientific-pdf-reader` — **科学PDF阅读**: 解析科研PDF中的正文、公式、表格和图像
- `citation-network-analyzer` — **引文网络分析**: 分析引用、共引、作者、机构和研究谱系
- `frontier-trend-detector` — **科研前沿识别**: 识别快速增长的主题、方法和关键团队
- `contradiction-finder` — **文献矛盾发现**: 定位不同研究之间的结论冲突及条件差异
- `method-extractor` — **方法提取**: 提取可复现实验或计算流程
- `parameter-extractor` — **参数提取**: 提取温度、压力、时间、浓度、模型与软件参数
- `claim-evidence-linker` — **结论证据关联**: 将论断关联到原文、图表和数据位置
- `reference-integrity-checker` — **参考文献完整性**: 检查伪造、错配、缺失和无支撑引用
- `literature-deduplicator` — **文献去重**: 处理预印本、正式论文和重复数据库记录
- `research-knowledge-graph` — **科研知识图谱**: 建立作者—材料—方法—性能—证据关系
- `semantic-indexer` — **科学语义索引**: 建立向量、关键词和结构化混合索引
- `knowledge-conflict-resolver` — **知识冲突消解**: 对相互冲突的结论保留条件、证据和版本
- `weekly-literature-briefing` — **文献简报**: 生成周期性前沿论文与重点变化简报
