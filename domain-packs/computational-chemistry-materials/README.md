# Computational Chemistry and Materials

## Scope

This pack covers electronic structure, atomistic mechanisms, condensed phases and materials-property prediction. It plans, validates and interprets work; it never reports an external solver, instrument or plant action as executed without an execution artifact.

## Operating sequence

1. Define the target observable and decision boundary.
2. Select the least-complex adequate model.
3. Record assumptions, units, parameter provenance and uncertainty.
4. Check conservation, limiting behaviour and numerical convergence.
5. Validate at the same measurement boundary as the experiment or plant.
6. Hand real execution to an approved environment.

## Method families

- DFT and wave-function methods
- transition-state and reaction-path analysis
- ab initio molecular dynamics
- periodic solids and surfaces
- solvation, free energy and enhanced sampling
- surrogate models with domain checks

## Required outputs

- machine-readable input and method contract
- convergence and validation record
- uncertainty and limitation statement
- provenance-linked artifacts and figures
- explicit human approval points
