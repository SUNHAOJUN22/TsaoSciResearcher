# Catalysis, Polymers and Composites

## Scope

This pack covers catalyst–reaction–chain architecture–mesostructure–processing–property links. It plans, validates and interprets work; it never reports an external solver, instrument or plant action as executed without an execution artifact.

## Operating sequence

1. Define the target observable and decision boundary.
2. Select the least-complex adequate model.
3. Record assumptions, units, parameter provenance and uncertainty.
4. Check conservation, limiting behaviour and numerical convergence.
5. Validate at the same measurement boundary as the experiment or plant.
6. Hand real execution to an approved environment.

## Method families

- quantum chemistry and microkinetics
- statistical thermodynamics and kinetic Monte Carlo
- polymerization kinetics and population balances
- molecular simulation and coarse graining
- rheology, crystallization and morphology characterization
- reactor/process modelling and uncertainty propagation

## Required outputs

- machine-readable input and method contract
- convergence and validation record
- uncertainty and limitation statement
- provenance-linked artifacts and figures
- explicit human approval points
